# Friend Management Client

This is a simple React-based web application for managing a list of friends. The app allows users to add, view, update, and delete friend information through a clean and straightforward interface.

This project is part of my practice while learning full-stack development. I built this to improve my React skills and to understand how a frontend application connects with a backend. At the same time, I am learning Node.js and using this project to practice backend integration and CRUD operations.

## Features

- Add a new friend  
- View all friends  
- Update existing friend details  
- Delete a friend  
- Simple and user-friendly interface  
- Connected with backend API  

## Technologies Used

- React  
- JavaScript (ES6+)  
- React Router  
- Tailwind CSS (if used)  
- Fetch API / Axios  

## Getting Started

To run this project locally:

    git clone https://github.com/MdBelayet3/friend-management-client.git
    cd friend-management-client
    npm install
    npm run dev

My backend Link https://github.com/MdBelayet3/friend-management-server

## Project Purpose

The main goal of this project is to practice CRUD operations and understand how frontend and backend work together. It also helps me learn Node.js by building and connecting APIs.
